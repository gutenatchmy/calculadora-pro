# 🧮 Calculadora Pro

Calculadora web interactiva desarrollada con HTML, CSS y JavaScript.

## 🚀 Demo
https://gutenatchmy.github.io/calculadora-pro

## ⚙️ Funcionalidades
- Operaciones matemáticas básicas
- Soporte de teclado
- Historial de operaciones
- Validación de errores

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript

## 📌 Objetivo
Este proyecto fue desarrollado como práctica para mejorar la lógica en JavaScript y la creación de interfaces interactivas.
